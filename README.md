AI Code Flow Mapper

AI Code Flow Mapper is a React + FastAPI based tool that analyzes source code and generates detailed insights.

#Features

* Multi-language support (Python, Java, C, C++)
* Function Detection
* Class Detection
* Import Analysis
* Viva Questions Generator
* Project Tree Generation
* Flowchart Generation
* Code Complexity Analysis
* Code Quality Score
* PDF Report Download
* AI Chatbot Assistant

## Tech Stack

# Frontend

* React
* Vite
* JavaScript

# Backend

* FastAPI
* Python
* AST Parser

## Installation

# Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

# Frontend

```bash
cd frontend
npm install
npm run dev
```
